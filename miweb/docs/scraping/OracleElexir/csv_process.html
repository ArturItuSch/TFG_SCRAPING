scraping.OracleElexir.csv_process

csv_process.py

Este módulo gestiona todo el procesamiento de archivos CSV descargados desde la API de Google Drive (Oracle's Elixir), convirtiéndolos en datos estructurados para su posterior almacenamiento en la base de datos del sistema.

Funcionalidades principales

  • Filtrado y organización de CSVs por liga y año mediante filtrar_ligas_automaticamente().
  • Limpieza y eliminación de archivos irrelevantes con borrar_csv_2025_lol_esports().
  • Extracción y mapeo de IDs únicos para equipos, jugadores y partidas a través de funciones como:
    • obtener_o_crear_id()
    • reemplazar_ids()
  • Transformación de CSVs crudos en objetos de alto nivel como:
    • Series (extract_all_series_and_partidos)
    • Jugadores en partida (extract_all_jugadores_en_partida)
    • Picks y Baneos (extraer_all_baneos_picks)
    • Objetivos neutrales (extract_objetivos_neutrales_matados)
    • Splits, Equipos y Jugadores (extract_all_splits, extract_all_teams, extract_all_players)

Estructura de carpetas esperada

  • /Resources/CSV/LEC/YYYY/*.csv: CSVs filtrados y organizados por año.
  • Diccionarios auxiliares (*.json) para mantener IDs consistentes y trazables entre ejecuciones:
    • player_ids.json
    • taem_ids.json
    • partidos_ids.json

Objetivo

Transformar grandes volúmenes de datos de partidas en estructuras limpias, normalizadas y listas para ser importadas en modelos Django relacionados con splits, partidos, jugadores y estadísticas específicas.

Este script es clave para convertir datos externos crudos en conocimiento estructurado dentro del sistema.

 84def filtrar_ligas_automaticamente(carpeta_csv, carpeta_salida_base, project_root):
 85    """
 86    Filtra automáticamente los archivos CSV por liga (por ejemplo, LEC) y los organiza por año y grupo.
 87
 88    Esta función lee todos los CSV en una carpeta base, identifica la liga (`league`) dentro del contenido,
 89    aplica equivalencias personalizadas (como agrupar 'EU LCS' y 'LEC' bajo 'LEC'), y guarda archivos
 90    filtrados por grupo y año en subcarpetas estructuradas.
 91
 92    Estructura de salida generada:
 93        carpeta_salida_base/
 94            LEC/
 95                2023/
 96                    2023_nombreoriginal_LEC.csv
 97                2024/
 98                    ...
 99
100    Args:
101        carpeta_csv (str): Carpeta donde se encuentran los CSV originales descargados.
102        carpeta_salida_base (str): Carpeta base donde se guardarán los archivos CSV filtrados y organizados.
103        project_root (str): Ruta raíz del proyecto usada para calcular rutas relativas.
104    """
105    os.makedirs(carpeta_salida_base, exist_ok=True)
106
107    equivalencias_ligas = {
108        'EU LCS': 'LEC',
109        'LEC': 'LEC',
110    }
111
112    archivos_csv = [
113        os.path.relpath(os.path.join(carpeta_csv, archivo), start=project_root)
114        for archivo in os.listdir(carpeta_csv)
115        if archivo.endswith('.csv') and os.path.isfile(os.path.join(carpeta_csv, archivo))
116    ]
117
118    for archivo in archivos_csv:
119        ruta_absoluta = os.path.join(project_root, archivo)
120        try:
121            df = pd.read_csv(ruta_absoluta, dtype={2: str})
122
123            if 'league' in df.columns:
124                nombre_base = os.path.splitext(os.path.basename(archivo))[0]
125                anio = nombre_base[:4] if nombre_base[:4].isdigit() else 'unknown'
126
127                # Obtener las ligas únicas reales
128                ligas_reales = df['league'].unique()
129
130                # Crear un diccionario para agrupar ligas por equivalencia
131                ligas_agrupadas = {}
132                for liga_real in ligas_reales:
133                    grupo = equivalencias_ligas.get(liga_real, liga_real)
134                    if grupo not in ligas_agrupadas:
135                        ligas_agrupadas[grupo] = []
136                    ligas_agrupadas[grupo].append(liga_real)
137
138                # Ahora iteramos por grupos (por ejemplo, 'LEC') y procesamos todos los archivos que pertenezcan a esos nombres originales
139                for grupo, ligas_del_grupo in ligas_agrupadas.items():
140                    datos_filtrados = df[df['league'].isin(ligas_del_grupo)]
141
142                    if not datos_filtrados.empty:
143                        carpeta_salida = os.path.join(carpeta_salida_base, grupo, anio)
144                        os.makedirs(carpeta_salida, exist_ok=True)
145
146                        nombre_archivo = f"{nombre_base}_{grupo}.csv"
147                        ruta_salida = os.path.join(carpeta_salida, nombre_archivo)
148
149                        datos_filtrados.to_csv(ruta_salida, index=False)
150                        print(f"Archivo guardado: {ruta_salida}")
151            else:
152                print(f"El archivo {archivo} no contiene la columna 'league'.")
153        except Exception as e:
154            print(f"Error procesando {archivo}: {e}")

156def cargar_diccionario(ruta):
157    """
158    Carga un diccionario desde un archivo JSON si existe, o retorna un diccionario vacío si no se encuentra.
159
160    Esta función es utilizada para mantener mapeos persistentes (como IDs de jugadores o equipos) entre
161    ejecuciones del script. También informa por consola la cantidad de elementos cargados o si se inicia vacío.
162
163    Args:
164        ruta (str): Ruta absoluta del archivo JSON que se desea cargar.
165
166    Returns:
167        dict: Diccionario cargado desde el archivo, o uno nuevo si no existe.
168    """
169    if os.path.exists(ruta):
170        with open(ruta, 'r', encoding='utf-8') as f:
171            dic = json.load(f)
172            print(f"Diccionario cargado de {ruta} con {len(dic)} entradas.")
173            return dic
174    print(f"No existe el archivo {ruta}, se crea diccionario vacío.")
175    return {}

201def es_uuid(valor):
202    """
203    Verifica si un valor dado es un UUID válido en formato estándar.
204
205    Args:
206        valor (str): Valor a comprobar.
207
208    Returns:
209        bool: `True` si el valor es un UUID válido, `False` en caso contrario.
210    """
211    if not isinstance(valor, str):
212        return False
213    patron_uuid = re.compile(
214        r'^[a-f0-9]{8}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{12}$', re.I
215    )
216    return bool(patron_uuid.match(valor))

218def extraer_hash_o_uuid(valor):
219    """
220    Extrae un UUID o hash válido desde un valor de entrada complejo.
221
222    Este método acepta entradas como UUIDs, hashes de 32 caracteres hexadecimales o strings con formato tipo
223    `"oe:player:abcdef1234567890..."`, extrayendo la parte identificadora.
224
225    Args:
226        valor (str): Valor original que puede contener un UUID o hash.
227
228    Returns:
229        str or None: UUID o hash extraído si es válido, o `None` si no se pudo interpretar.
230    """
231    if not isinstance(valor, str):
232        return None
233    valor = valor.strip()
234
235    if es_uuid(valor):
236        return valor
237
238    if ":" in valor:
239        partes = valor.split(":")
240        posible_hash = partes[-1]
241        if re.fullmatch(r"[a-fA-F0-9]{32}", posible_hash):
242            return posible_hash
243
244    if re.fullmatch(r"[a-fA-F0-9]{32}", valor):
245        return valor
246
247    return None

249def obtener_o_crear_id(diccionario, valor_original):
250    """
251    Obtiene un ID único basado en un hash/UUID válido extraído del valor original.
252    Si no existe un ID asociado, genera un nuevo UUID persistente y lo guarda en el diccionario.
253
254    Esta función garantiza que cada identificador (jugador, equipo, partida) se mantenga constante
255    entre múltiples ejecuciones, incluso si el valor original es no estándar o cambiante.
256
257    Args:
258        diccionario (dict): Diccionario donde se almacenan los mapeos entre valores originales y UUIDs.
259        valor_original (str): Valor fuente del cual se intentará extraer un hash o UUID válido.
260
261    Returns:
262        str: UUID asignado o recuperado desde el diccionario para el valor proporcionado.
263    """
264    id_base = extraer_hash_o_uuid(valor_original)
265
266    if id_base is None:
267        # Aquí usamos el valor original como clave para mapear un UUID persistente
268        if valor_original not in diccionario:
269            nuevo_uuid = str(uuid.uuid4())
270            diccionario[valor_original] = nuevo_uuid
271            print(f"Clave no estándar '{valor_original}' añadida con UUID: {nuevo_uuid}")
272        return diccionario[valor_original]
273
274    if es_uuid(valor_original):
275        return valor_original
276
277    if id_base not in diccionario:
278        nuevo_uuid = str(uuid.uuid4())
279        diccionario[id_base] = nuevo_uuid
280        print(f"Añadido nuevo mapeo: {id_base} -> {nuevo_uuid}")
281
282    return diccionario[id_base]

284def obtener_rutas_csv(carpeta):
285    """
286    Busca de forma recursiva rutas absolutas a archivos CSV dentro de subdirectorios anidados por año.
287
288    Esta función asume que los archivos CSV están organizados en carpetas por año, por ejemplo:
289    `carpeta/2023/archivo.csv`, `carpeta/2024/archivo.csv`, etc.
290
291    Args:
292        carpeta (str): Ruta de la carpeta raíz que contiene subdirectorios con archivos CSV.
293
294    Returns:
295        list[str]: Lista ordenada de rutas absolutas a los archivos CSV encontrados.
296    """
297    rutas_csv = []
298    for year in os.listdir(carpeta):
299        year_path = os.path.join(carpeta, year)
300        if os.path.isdir(year_path):
301            for archivo in os.listdir(year_path):
302                if archivo.endswith('.csv'):
303                    archivo_path = os.path.join(year_path, archivo)
304                    rutas_csv.append(archivo_path)
305    rutas_csv.sort() 
306    return rutas_csv

308def reemplazar_ids(df, columna, diccionario):
309    """
310    Reemplaza los valores de una columna en un DataFrame por UUIDs únicos utilizando un diccionario de mapeo.
311
312    Para cada valor original en la columna especificada, se obtiene o genera un UUID a través del diccionario.
313    Este procedimiento permite mantener la consistencia de identificadores únicos a lo largo de múltiples ejecuciones.
314
315    Args:
316        df (pd.DataFrame): DataFrame que contiene la columna a transformar.
317        columna (str): Nombre de la columna cuyos valores serán reemplazados.
318        diccionario (dict): Diccionario que almacena los mapeos entre valores originales y UUIDs.
319    """
320    if columna not in df.columns:
321        return
322    nuevos_ids = []
323    for val in df[columna]:
324        nuevos_ids.append(obtener_o_crear_id(diccionario, val))
325    df[columna] = nuevos_ids

362def extract_all_splits():
363    """
364    Extrae y agrupa todos los splits únicos presentes en los archivos CSV de partidas de la LEC.
365
366    Esta función recorre los archivos CSV organizados por año y liga, filtra los datos por el año
367    correspondiente al archivo, y construye un diccionario con información unificada de cada split,
368    incluyendo su identificador, liga, año y nombre de la temporada.
369
370    Returns:
371        dict[str, dict]: Diccionario con claves `split_id` y valores que contienen metainformación
372        del split: ID, liga, año y nombre de la temporada (season).
373    """
374    all_splits = {}
375
376    rutas_csv = obtener_rutas_csv(CARPETA_CSV_LEC)
377
378    for csv_file_path in rutas_csv:
379        try:
380            nombre_archivo = os.path.basename(csv_file_path)
381            year_from_filename = nombre_archivo.split('_')[0]
382            df = pd.read_csv(csv_file_path)
383            if 'year' in df.columns:
384                df = df[df['year'].astype(str) == year_from_filename]
385            else:
386                continue
387
388            if df.empty:
389                continue
390            df['split'] = df['split'].fillna('unknown')
391            splits = df['split'].drop_duplicates()
392            year = os.path.basename(csv_file_path).split('_')[0]
393            league = df['league'].iloc[0] if 'league' in df.columns else 'unknown'
394
395            for split in splits:
396                split = str(split).strip().lower() if split else 'unknown'
397                split_id = f"{year}{split}"
398
399                if split_id not in all_splits:
400                    all_splits[split_id] = {
401                        "id": split_id,
402                        "league": league,
403                        "year": year,
404                        "season": split
405                    }
406
407        except Exception as e:
408            print(f"Error al procesar el archivo {csv_file_path}: {e}")
409
410    return all_splits

412def extract_all_series_and_partidos():
413    """
414    Extrae todas las series y partidos a partir de los archivos CSV de partidas de la LEC.
415
416    Esta función agrupa las partidas por `gameid` y organiza las series con base en la aparición
417    del juego 1 (`game == 1`). Cada serie contiene metainformación como fecha, número de partidas,
418    parche, y si corresponde a playoffs. Además, se extraen los datos detallados de cada partido
419    dentro de cada serie mediante `extract_partidos_de_serie`.
420
421    Returns:
422        tuple[dict, dict]:
423            - dict con claves `serie_id` y valores con metadatos de la serie (split_id, fecha, patch, playoffs).
424            - dict con claves `gameid` y valores con datos del partido (orden, duración, equipos, ganador, etc.).
425    """
426    all_series = {}
427    all_partidos = {}
428
429    rutas_csv = obtener_rutas_csv(CARPETA_CSV_LEC)
430    serie_counters = {}
431
432    for csv_file_path in rutas_csv:
433        try:
434            df = pd.read_csv(csv_file_path)
435            df = df.reset_index(drop=True)
436
437            current_serie_id = None
438            current_serie_partidos = []
439            last_gameid = None
440
441            for idx, row in df.iterrows():
442                gameid = row['gameid']
443                game = row['game']
444                year = os.path.basename(csv_file_path).split('_')[0]
445                split = str(row['split']).strip().lower()
446                patch = row.get('patch')
447                fecha_completa = row.get('date')
448                fecha_solo_dia = None
449                playoffs = bool(row.get('playoffs', 0))
450
451                if pd.notna(fecha_completa):
452                    try:
453                        fecha_solo_dia = datetime.strptime(fecha_completa, '%Y-%m-%d %H:%M:%S').date()
454                    except ValueError:
455                        print(f"❌ Formato de fecha inválido: {fecha_completa}")
456
457                if (gameid != last_gameid and game == 1) or current_serie_id is None:
458                    # Guardar la serie anterior
459                    if current_serie_id and current_serie_partidos:
460                        all_series[current_serie_id] = {
461                            'split_id': f"{year}{split}",
462                            'num_partidos': len(current_serie_partidos),
463                            'patch': patch,
464                            'dia': fecha_solo_dia,
465                            'playoffs': playoffs
466                        }
467
468                        df_serie = df[df['gameid'].isin(current_serie_partidos)]
469                        partidos_en_serie = extract_partidos_de_serie(current_serie_id, df_serie)
470                        all_partidos.update(partidos_en_serie)
